# DeFi Swap

## Local Development

The following assumes the use of `node@>=10`.

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`

## Steps to deploy
`yarn compile`  
`yarn truffle-compile`  
`yarn replace-factory`  
`yarn truffle-migrate` OR `yarn truffle-migrate-ropsten`
`yarn truffle-migrate-c4ei`
